# Social Spotify

A tool for friendly music lovers to share, rate, and comment on songs and playlists using the Spotify API.

# Technologies Used
* React.js
* Spotify API
* OAuth2.0
* Express.js
* MongoDB
* Material-UI
* JavaScript
* HTML5
* CSS3

# Usage
```
git clone https://github.com/jjnotjayjay/social-spotify.git
cd social-spotify
npm i
npm run watch
npm run watch-server
```
